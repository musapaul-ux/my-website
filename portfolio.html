My first project

In this project, I developed a simple banking application that helps enhance banking activities and helps import efficiency.
The application supports the following basic operations

  • Checking acccount balance
  • Depositing money
  • withdrawing money

This application was entirely developed in C++, though I can also work with different other programming languages like;
  • java
  • Python
  • C

The idea of this project came to the challenges in the Banking systems in several banks. After a through research, I found out that, most of the banks face a problem of long lines at the ATM teller. DEveloping this application will enable users of these banks to transact at ease without the unnecessary long waiting lines.
