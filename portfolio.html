My first project

In this project, I developed a Rentals managing system that helps enhance tenants in booking for rooms online and helps import efficiency. The application supports the following basic operations

  • Displaying available rooms
  • Displaying Room sizes
  • Displaying there Prices
  • Collect tenants details
  • Allows customers to place their bookings
  • Calculate the total bill
  • Displays a successfuly message when order is placed

This application was entirely developed in Java, though I can also work with different other programming languages like;
  • C++
  • Python
  • C
  • HTML
